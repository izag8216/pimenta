# Contributing to Pimenta

Thank you for your interest in contributing!

## Development Setup

1. Fork and clone the repository
2. Open `index.html` in your browser
3. That's it - no build tools required

## Making Changes

- All game code is in `index.html` (single-file architecture)
- Keep the zero-dependency philosophy
- Test in Chrome, Firefox, and Safari
- Ensure the game runs at 60fps

## Submitting Changes

1. Create a feature branch
2. Make your changes
3. Test thoroughly in multiple browsers
4. Submit a pull request with a clear description

## Code Style

- Vanilla JavaScript (no TypeScript, no frameworks)
- Keep the single-file architecture
- Use clear variable names
- Comment complex game logic

## Reporting Issues

- Include browser version and OS
- Describe steps to reproduce
- Include any console errors
